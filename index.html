How To Contribute

This project was created by Matthew Trask contributions from the community. If you need help, hit Matt up on Twitter.

How To Add Someone in a few easy steps:

  • Step One: Go to the repository here. You will need an account.
  • Step Two: On a new branch, navigate to the _data folder, and edit the politicians.yml file.
  • Step Three: Open a Pull Request and tag Matt (@matthewtrask) who will promptly merge it.
